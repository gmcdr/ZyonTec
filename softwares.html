Cadastro Z



Sobre o Software :


CadastroZ é um Programa para cadastrar alunos em uma academia, podendo também cadastrar pagamentos, deletar e buscar alunos cadastrados, utilizando o banco SQLITE. Interface gráfica feita com tkinter. Software desenvolvido totalmente em Python !


A versão disponível para download é apenas para testes, para uma versão custumizada para o seu negócio, contate-nos.
